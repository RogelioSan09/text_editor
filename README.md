# Text Editor

## Description
An application that enables the user to create notes or code snippets with or without the use of an internet connection. The application automatically saves any content the user inputs within the text editor when the user clicks away from the DOM window. In addition the application allows the user to download their application onto their desktop.

## Usage
Automatically saves content inside the text editor when user clicks off the DOM window.

User can install the application as a progressive web application.
